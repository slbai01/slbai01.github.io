任务提交

快捷提交命令sbatcht <"commond">

  • 指定任务依赖
    • 依赖单个job：--dependency="afterok:1900100"
    • 依赖多个job：--dependency="afterok:1900100:1900101"
  • 提交任务便捷别名 alias sbatcht='sbatch -q normal -n 1 --job-name tmp1 -e slurm.err -o slurm.out --wrap '

修改排队任务参数

sudos control update MinCPUsNode=8 Partition=fat MinMemoryNode=10 JobId=1249 # MinMemoryNode 单位是 Mb
sudo scontrol update Priority=<number> JobId=xx # Only the Slurm administrator or root can increase job’s priority

可操控修改参数列表

恢复节点

sudo scontrol update NodeName=node3 State=RESUME

查看任务状态

scontrol show jobid=xx

终止任务

  • 终止单个任务 scancel jobid
