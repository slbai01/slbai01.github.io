SLURM常用命令

Posted on 2022-10-05 Edited on 2023-07-06 In linux进阶 Views:
Symbols count in article: 926 Reading time ≈ 1 mins.

任务提交

快捷提交命令sbatcht <"commond">

  • 指定任务依赖
    • 依赖单个job：--dependency="afterok:1900100"
    • 依赖多个job：--dependency="afterok:1900100:1900101"
  • 提交任务便捷别名 alias sbatcht='sbatch -q normal -n 1 --job-name tmp1 -e slurm.err -o slurm.out --wrap '

修改排队任务参数

sudos control update MinCPUsNode=8 Partition=fat MinMemoryNode=10 JobId=1249 # MinMemoryNode 单位是 Mb
sudo scontrol update Priority=<number> JobId=xx # Only the Slurm administrator or root can increase job’s priority

可操控修改参数列表

恢复节点

sudo scontrol update NodeName=node3 State=RESUME

查看任务状态

scontrol show jobid=xx

查询区间内基因信息

1. 获取对应基因组位置信息

如查询 AX-110604864, AX-111546862 位点内基因信息。

根据 55K 芯片信息，获取序列信息 (对于多态位点，任选其一)

在节节麦基因组网站进行blastn查询

查询结果为：

ArrayID AX-110604864 AX-111546862 - AX-110177050 AX-109104358
T093 Chr4 4113168 Chr4 3119203 - Chr3 19255786 Chr3 22041177
AK58 chr4D 4064411 chr4D 3599881 - chr3D 17829703 chr3D 20351356

2. 获取区间内基因信息

根据起始位置从 gff 文件内使用 awk 提取基因ID即可
如无编程基础，可在节节麦信息提取网站提取基因信息。
